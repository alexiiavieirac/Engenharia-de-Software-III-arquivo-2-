Principais Operadores Lógicos e Estruturas de Lógica de Programação

Operadores Lógicos

Operador Descrição Exemplo
AND (&&) Retorna verdadeiro se ambas as expressões forem verdadeiras. (x > 5) && (y < 10)
OR (||) Retorna verdadeiro se pelo menos uma das expressões for verdadeira. (x > 5) || (y < 10)
NOT (!) Inverte o valor lógico da expressão. !(x > 5)

Estruturas de Lógica de Programação

  • If / Else

    Permite a execução de um bloco de código se uma condição for verdadeira. O bloco else pode ser usado para executar um código alternativo se a condição for falsa.

    if (x > 5) {
    // código se x for maior que 5
} else {
    // código se x for menor ou igual a 5
}
  • Switch

    Executa um bloco de código entre várias opções baseadas no valor de uma variável.

    switch (x) {
    case 1:
        // código para x igual a 1
        break;
    case 2:
        // código para x igual a 2
        break;
    default:
        // código para qualquer outro valor de x
}
  • For

    Executa um bloco de código repetidamente com base em uma condição inicial, condição de parada e incremento/decremento.

    for (int i = 0; i < 10; i++) {
    // código a ser repetido
}
  • While

    Executa um bloco de código enquanto uma condição for verdadeira.

    while (x < 10) {
    // código enquanto x for menor que 10
}
  • Do While

    Similar ao while, mas garante que o bloco de código seja executado pelo menos uma vez.

    do {
    // código a ser repetido
} while (x < 10);
